# eloquent
Using ORM Eloquent

## Cloning
```
git clone https://github.com/ClubeDosGeeksCoding/eloquent.git
```

## Installing
```
cd eloquent
composer install
``` 

## Database
```
php artisan migrate
php artisan db:seed
```
